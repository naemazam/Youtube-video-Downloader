Python  based Youtube video Downloader GUI Application.   

 
## Installation Python Dependencies

Import pytube
```python
  pip install pytube
```
Import tkinter
```python
  pip install tkinter
```

# Download & Installation

Download 
```bash
git clone https://github.com/naemazam/Youtube-video-Downloader.git
```

Run Application 
```python
 python3 yt-downloader.py
```
